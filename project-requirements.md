# Требования к проекту (HTML + CSS + SCSS)

1. SCSS: использовать @mixin и переменные ($variables).
2. :root: использовать CSS-переменные через :root.
3. Flexbox: использовать flex для построения макета.
4. Grid: использовать CSS Grid (grid).
5. Media-запросы: адаптивность для экранов 1920 / 1024 / 768 / 320.
6. Анимации: использовать @keyframes, animation и эффекты hover.
7. Position: использовать relative / absolute / fixed / sticky.
8. Форма: минимум один набор элементов input / button / textarea.
9. Iframe: вставить YouTube-видео или Google Maps или интегрированную 3D-сцену.
10. Шрифты: подключить Google Fonts или использовать собственные шрифты.
11. Иконки: использовать Font Awesome или собственные SVG-иконки.
12. Изображения: предпочтительно PNG (оптимизированные, небольшого размера).
13. Popup по умолчанию (Default popup).
14. Минимум одна страница.
